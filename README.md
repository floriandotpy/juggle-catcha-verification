## CATCHA

A tiny web demo to toy with the idea of detecting juggling tricks (or juggling props initially) to use as a CAPTCHA mechanism (hence CATCHA, get it?).

## How to run

1. Find a copy of `yolov8n.onnx` online and put in `model/` directory.
2. Serve files through local web server: `python3 -m http.server 8000`
3. Open browser at `http://localhost:8000`
